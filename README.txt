Deskripsi
-------------------------------------------------------------------------------------------------
Website ini dibuat sebagai tugas Ujian Tengah Semester (UTS) mata kuliah Pemrograman Web I.
Isi website berupa Curriculum Vitae (CV online) dengan tampilan modern, responsive, dan memenuhi
seluruh ketentuan dosen:

- Foto diri (200x200 px)
- Minimal 2 paragraf deskripsi diri dengan text formatting (<b>, <i>, <u>, <mark>)
- Heading tags (<h1>, <h2>, <h3>)
- Komentar HTML
- Minimal 3 link internal atau eksternal
- Minimal 3 gambar 
- Minimal 2 tabel
- Minimal 2 list

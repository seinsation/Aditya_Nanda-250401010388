Halo! Saya Aditya Nanda

Saya adalah seorang Staff IT dan Administrasi yang saat ini sedang menempuh pendidikan S1 di Universitas Siber Asia (UNSIA), Program Studi Informatika secara daring (PJJ). Saya memiliki pengalaman lebih dari empat tahun bekerja di rumah sakit, di mana saya terbiasa mengelola data, membuat laporan, dan memastikan akurasi data sesuai standar operasional.

Saya adalah pribadi yang teliti, disiplin, dan bertanggung jawab. Saya menguasai Microsoft Excel dan Microsoft Word, serta memiliki kemampuan troubleshooting dasar komputer dan jaringan. Saya percaya bahwa semangat belajar adalah kunci untuk terus berkembang, dan saat ini saya aktif mempelajari pemrograman web sebagai bagian dari perjalanan akademis saya di UNSIA.

Saya berkomitmen untuk terus belajar hal-hal baru dan siap berkontribusi secara maksimal dalam tim maupun secara mandiri.
